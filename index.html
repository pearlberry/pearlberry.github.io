Phone Menu System Visualizer

Spring 2015​ - I was a part of a team that was tasked with creating an in-browser IVR visualizer that would allow users to view a graphical “tree-like” representation of their phone menu system. We worked with the company’s existing APIs, using HTML, CSS, and JavaScript to create this. We used a JavaScript diagramming library called JointJS to display the elements. The company will use what we created in their new, more user-friendly system for managing phone systems. My main responsibilities included project management, issue tracking, testing, and front-end development.


DrinkFargo

Fall 2014​ - I worked with another person to create a web-based application for finding drink specials at bars in Fargo. Our main focus of this project was usability, and we conducted tests with real users to get a feel for user flow. A user could search for drink specials based on Bar Name, Drink Type, or Bar Type (ex. Irish themed bar, or Western themed bar). They could also search for Daily Specials and it would display drink specials specific to the current day. My main responsibilities included MySQL database implementation, queries, and integration with the site, and HTML/CSS development.


Cactus

Spring 2014 ​- Our team created a side-scrolling game for Windows Phone, inspired by existing side-scrolling games like ‘Helicopter’ and ‘Jetpack Joyride’. The goal of the game is to ‘fly’ as far as possible without hitting obstacles, while collecting coins along the way to increase your score. We wrote this game using C#, and created all elements of the game including sprites, backgrounds, and menu screens. My main responsibilities included project management - writing documentation and progress reports, content (sprites, menu screens, backgrounds) creation using Photoshop, and feature development using C# - collision detection, score counting, and parallax background motion.
