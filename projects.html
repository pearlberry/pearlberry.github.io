Projects

Here are some of the projects I've worked on as a part of a team and individually! For more information on what I've done, contact me.

Team Project

Cactus

Our team created a side-scrolling game, inspired by existing games like ‘Helicopter’ and ‘Jetpack Joyride’. The goal of the game is to ‘fly’ as far as possible without hitting obstacles, while collecting coins along the way to increase your score. We wrote this game using C#, and created all elements of the game including sprites, backgrounds, and menu screens. My main responsibilities included project management - writing documentation and progress reports, content (sprites, menu screens, backgrounds) creation using Photoshop, and feature development using C# - collision detection, score counting, and parallax background motion.

Team Project

Phone Menu System Visualizer

Member of a team that was tasked with creating an in-browser IVR visualizer that would allow users to view a graphical “tree-like” representation of their phone menu system. Worked with the company’s existing APIs, using HTML, CSS, and JavaScript to create this. Used a JavaScript diagramming library called JointJS to display the elements. The company will use what we created in their new, more user-friendly system for managing phone systems. My main responsibilities included project management, issue tracking, testing, and front-end development.

Pair Project

DrinkFargo

Worked with another person to create a web-based application for finding drink specials at bars in Fargo. Our main focus of this project was usability, and we conducted tests with real users to get a feel for user flow. A user could search for drink specials based on Bar Name, Drink Type, or Bar Type (ex. Irish themed bar, or Western themed bar). They could also search for Daily Specials and it would display drink specials specific to the current day. My main responsibilities included MySQL database implementation, queries, and integration with the site, and HTML/CSS development.

Individual Project

Industrial Park Website

In the process of working with a real estate developer to create an informational website for a new industrial park in Bismarck. Created a rough design for the site using Bootstrap. The requirements for the site have been discussed with the developer.
